# Glosario (corto)

- **Scene (Escena):** un “nivel” o pantalla del juego.
- **GameObject:** objeto dentro de la escena.
- **Component:** pieza que se agrega a un GameObject para darle funciones.
- **Prefab:** un objeto guardado como “plantilla” para reutilizar.
- **Collider:** define choques.
- **Trigger:** un collider que detecta entrada/salida sin empujar físicamente.
- **Script:** código (C#) que controla comportamientos.
- **Inspector:** panel donde editas componentes.
- **Hierarchy:** lista de objetos en la escena.
